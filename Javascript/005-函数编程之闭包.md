## 什么是闭包

闭包是一种能够访问**自由变量**的函数。

**什么是自由变量？**
自由变量，不是函数的参数，也不是函数内部的变量，是函数外部的变量。

**什么函数才算闭包？**

- 即使创建它的上下文已经销毁，它仍然存在，比如内部函数从父函数中返回；
- 在代码中引用了自由变量；

**闭包的作用：**

1. 在函数的外部访问函数内部的局部变量；
2. 保存某个函数当前的执行上下文变量；

**闭包的缺点：**
可能会造成内存泄漏；

### 闭包必刷题

```js
var data = [];

for (var i = 0; i < 3; i++) {
	data[i] = function () {
		console.log(i);
	};
}

data[0]();
data[1]();
data[2]();
```

使用闭包来保存当前执行的变量：

```js
var closure2 = function () {
	var data = [];
	// 使用闭包能够保存变量
	for (var i = 0; i < 3; i++) {
		data[i] = (function (i) {
			return function () {
				console.log(i);
			};
		})(i);
	}
	data[0]();
	data[1]();
	data[2]();
};
```
